Welcome to MAX.ai documentation!
===================================
Max.ai provides a ready list of utilities to create end to end ML pipelines and solve complex business problems using state of the art ML modules. Max.ai comes equipped with utilities to store, process and transform data, automated feature engineeriing and a host of ML models - Classification, Clustering, Regression, Time-series, NLP and much more. Check out the :doc:`quickstart` section for jumping directly into a working example. You can refer to how to :ref:`installation` to install max.ai on your project.
.. note::

   This project is under active development.
